# QuickSigns - Installation Guide

## Requirements
- **Blender 4.2 or higher** (uses the new Extension system)
- Internet connection
- FREE Google Fonts API key

## Quick Install (3 steps)

### 1. Download
Download `quick_signs.zip` from the release

### 2. Install Extension in Blender

**For Blender 4.2+:**
1. Open Blender
2. Go to `Edit` → `Preferences` → `Get Extensions`
3. Click the dropdown menu (⌄) in the top right
4. Select `Install from Disk...`
5. Select `quick_signs.zip` (don't unzip it!)
6. Click `Install from Disk`

**Alternative (Legacy Add-ons method):**
1. Go to `Edit` → `Preferences` → `Add-ons`
2. Click `Install...` button (top right)
3. Select `quick_signs.zip`
4. Click `Install Add-on`

### 3. Enable Extension
1. The extension should be enabled automatically
2. If not, search for "QuickSigns" and enable it
3. Press `N` in 3D Viewport to see the `Signs` tab

## First Time Setup

### Get Your FREE Google Fonts API Key

1. Go to https://console.cloud.google.com/
2. Create a new project (or select existing)
3. Search for "Google Fonts Developer API" and enable it
4. Go to "Credentials" → "Create Credentials" → "API Key"
5. Copy the API key

### Enter API Key in Blender

1. Press `N` in the 3D Viewport to open the sidebar
2. Click the `Signs` tab
3. Paste your API key in the "API Key" field
4. You're ready to go!

## Usage

1. **Search Fonts**:
   - Select a style filter (Handwritten, Vintage, etc.)
   - Optionally type a search term
   - Click "Search Fonts"

2. **Preview Fonts**:
   - Enable "Auto Preview" to see fonts as you select them
   - Or click "Manual Preview" to preview selected font

3. **Create Sign**:
   - Click a font to select and download it
   - Enter your text (or use the random name generator)
   - Adjust size, depth, bevel, and material
   - Click "Create Sign"

## Troubleshooting

**"Please enter your Google Fonts API key"**
- Make sure you've pasted your API key in the Signs panel

**Add-on doesn't appear in the list**
- Make sure you installed the ZIP file (not the extracted folder)
- Try restarting Blender

**No fonts appearing after search**
- Check your internet connection
- Verify your API key is correct
- Try changing the style filter to "All Styles"
